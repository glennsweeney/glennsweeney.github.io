# glennsweeney.github.io

## Environment

1. Install [nvm](https://github.com/nvm-sh/nvm)

1. Install node:

    ```bash
    nvm install node
    ```

## Project Setup

1. Start a SvelteKit project ([docs](https://kit.svelte.dev/docs/creating-a-project))

## Developing

1. Install dependencies:

    ```bash
    npm install
    ```

2. Start the development server:

    ```bash
    npm run dev -- --open
    ```

## Building

### Preview Version

```bash
npm run preview
```

### Production Version

```bash
npm run build
```
